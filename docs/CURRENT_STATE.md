This empirically confirms the day-one tiering: 8B-class for fast smoke tests, 32B-class as the interactive workhorse, 70B-class reserved for "willing to wait 60-90 seconds for a deeper answer" use cases, and (once GPU #3 is unrestricted) 70B Q6 across all 3 GPUs as the highest-quality tier on this hardware.
